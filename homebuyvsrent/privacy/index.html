Home Buy vs Rent Privacy Policy

Summary

Home Buy vs Rent primarily stores your comparison data on your device. The current codebase reviewed for this disclosure does not appear to include third-party ad SDKs or behavioral advertising trackers.

Your data is sent off-device only for features you choose to use, including Apple Maps address autocomplete, optional AI-generated estimates, optional AI listing extraction from images, temporary CloudKit share codes, and feedback emails you decide to send.

If you do not use AI, sharing, or feedback features, the app mostly operates with local storage and Apple-provided address lookup services.

At a Glance

Data stored locally

Home entries, household profile details, onboarding state, AI provider selections, selected models, currency preference, custom feature lists, and optional API logs.

Sensitive inputs

Addresses, household and workplace details, pricing assumptions, notes, uploaded listing screenshots or photos, and anything you send in prompts or support emails.

Cloud sharing

If you create a share code, the selected homes are uploaded to a public CloudKit database and are set to expire after about 7 days.

Information Home Buy vs Rent Collects

Information you enter

Home addresses, ZIP codes, household members, commute and workplace details, budget assumptions, rent or home price inputs, amenities, notes, and any feedback messages you write.

Images you choose

If you use the camera or photo-library import feature, the app processes listing screenshots or photos to extract property details. Those images are only transmitted when you intentionally use AI listing extraction.

Configuration data

Selected AI provider, chosen text and vision models, currency preference, and onboarding status are stored locally. API keys are stored in iOS Keychain rather than plain preferences.

Diagnostic and support data

If enabled, API request logs are stored locally. If you send feedback, the email can include device model, iOS version, app version, AI provider configuration, and attached API logs.

How Data Is Used

To build, compare, rank, and save housing scenarios.
To autocomplete and resolve addresses through Apple Maps services.
To provide optional AI-assisted housing estimates and analysis.
To extract listing details from photos or screenshots you choose to analyze.
To generate temporary share codes when you explicitly share selected homes.
To help diagnose bugs if you choose to send feedback with logs or device details.

When Data Leaves Your Device

Feature Data involved Recipient Notes
Address autocomplete Address fragments and selected place lookups Apple Maps / MapKit Used when you type an address or choose an autocomplete result.
AI estimates Prompt text built from your home and household inputs Anthropic, OpenAI, Perplexity, or Google Gemini depending on your selected provider Only used if you configure an API key and request AI-powered estimates.
Camera or screenshot listing scan Listing image plus extraction prompt Anthropic, OpenAI, or Google Gemini depending on your selected vision provider Only used when you run the listing extraction feature. Perplexity is not used for vision in the current app.
Model refresh or connection test Your API key and provider request metadata The selected AI provider Used when you test a connection or refresh available model lists in Settings.
Share code export Encoded home data, generated code, timestamps, and expiry time Apple CloudKit public database Current implementation sets share code records to expire after about 7 days.
Feedback email Your message, device and app metadata, and optional API log attachment Your configured mail provider and Velora App Labs support inbox Sent only if you choose to send feedback.

Storage and Retention

Most app data is stored locally on your device until you edit or delete it, or until you use the app's clear-data controls.

API keys are stored in Keychain. Other app settings and profile data are stored through local iOS app storage such as UserDefaults.

Shared home records created through CloudKit are intended to expire after about 7 days based on the current code path.

Your Choices

Use local mode

You can avoid AI features entirely by not adding API keys. The app then uses local heuristics instead of provider calls.

Clear saved data

The Settings screen includes a clear-all-data flow that resets homes, household profile, onboarding state, and AI keys.

Children and Security

Home Buy vs Rent is not directed to children under 13. Do not submit personal information through the app if you are under 13.

No app can guarantee absolute security. We rely on Apple platform protections, local device storage, Keychain storage for API keys, and the security practices of the external services you choose to use.
